Simple++ : Learning C++ from the ground up

Hi there, I'm NotAPenguin, and welcome to my C++ tutorials. These tutorials are aimed at everyone who wants to learn C++. C++ is a language that is often seen as very difficult to learn. These tutorials are written with the opposite in mind: C++ should be an easy language to learn. You will also learn about good and back practices, to avoid common pitfalls when writing C++ code.
